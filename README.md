# رادار الأسهم السعودية 📈

أداة تحليل أسهم تداول مع تنبيهات إيميل تلقائية.

## الملفات
- `api/stocks.js` — سيرفر يجلب بيانات Yahoo Finance
- `public/index.html` — واجهة المستخدم
- `vercel.json` — إعدادات النشر

## النشر على Vercel
1. ارفع المشروع على GitHub
2. اذهب لـ vercel.com وسجّل دخول بـ GitHub
3. اضغط "New Project" واختر هذا الـ repo
4. اضغط Deploy — خلاص!
